Lucky 7

Read an integer number n. This is a number of integer values you will receive on the next n lines. Among these numbers find the multiples of 7 (that is, numbers that could be divided by 7 without the remainder) and print their squares. Each square number should be printed on a new line.

You can write n to some variable and then call input() function exactly n times using for loop.

You can either output each square immediately after reading the input with the number, or you can calculate all the squares at first and then output them. It doesn't matter which option you choose: the only thing that matters is that you should output m lines with correct squares.


Sample Input:
7
8
11
-49
0
3564
14
33

Sample Output:
2401
0
196



Memory limit: 256 MB
Time limit: 15 seconds
