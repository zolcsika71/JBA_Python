Latin equivalents

Imagine you're working at IMDb and your new task is to write a program that will replace all the letters with ligatures and diacritic marks with their equivalents from the Latin alphabet in the celebrities' names. The list of replacements is the following:

é -> e
ë -> e
á -> a
å -> a
œ -> oe
æ -> ae

Complete the code below so that the function would take a string with diacritics and return the one where all of them have been replaced with the equivalents.


Sample Input:
Charlotte Brontë

Sample Output:
Charlotte Bronte



Memory limit: 256 MB
Time limit: 15 seconds
