Tourists

Eugene and Rose decided to travel abroad. For convenience, they will go to a country where one of them has already been, but not where they both were.

Output a set of potential countries for them.

If you feel up to trying something new, use set.symmetric_difference(other) that returns a new set that contains all items from both sets, except for their common elements.


Sample Input:
Greece Netherlands Colombia UK
Italy UK Russia Greece Canada

Sample Output:
{'Colombia', 'Russia', 'Italy', 'Netherlands', 'Canada'}
